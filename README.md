1. git clone:

````
git clone https://github.com/CapstoneProjectDBS/CapstoneProjectDBS.git
````

2. Masuk ke direktori backend
````
cd backend
````

3. Instal node module
````
npm install
````

4.Copy env
````
cp .env.example .env
````

5. Jalankan server + migrasi otomatis
````
node index.js
````

6. Masuk ke direktori frontend
````
cd frontend
````

7. Instal node module
````
npm install
````

8. Jalankan frontend
````
npm run dev
````
